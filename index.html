Open Graph Image Generator for daneden.me

This tiny website serves a single purpose as a way to generate Open Graph images for my website, daneden.me.

The images are generated by the api route on this website with a title parameter, like so:

<meta name="og:image" content="https://daneden-og-generator.daneden.now.sh/api?title=An%20Example%20Generated%20Image"/>
